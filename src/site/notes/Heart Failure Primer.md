# Heart Failure Primer

Heart failure is a clinical syndrome of impaired cardiac function with reduced output and/or elevated filling pressures.

## Key Points

- Common causes: ischemic heart disease, hypertension, valvular disease.
- Typical symptoms: dyspnea, orthopnea, edema.
- Useful tests: BNP/NT-proBNP, echocardiography.
